DC Deck-Building Game Tracker

Track every game, every hero, every nemesis defeated.

What is this?

This is a personal tracker for the DC Deck-Building Game by Cryptozoic Entertainment. Log your sessions, track which heroes you play, how many nemesis cards you defeat, and see your performance over time — all saved locally in your browser.

Whether you're playing a base set, an expansion crossover, or Crisis mode, this tool keeps a detailed record so you can look back at your full play history and stats.

Compatibility note: This tracker supports the standard competitive DC Deck-Building Game format. It does not yet support solo play or the Rebirth DC Deck-Building Game.

How to use it

First time? Go to ⚙️ Settings and set up your player names. You can add base games, crossovers, and oversized cards there too. Batman and Superman are included by default to get you started.

Then head to 🎮 Add Game to log each session. Select your base game, crossover, date, players, and oversized cards — the app handles the rest.

⚙️ Settings

Active Players

▼

Your player roster — these names appear in the Add Game form. Min 2, max 5 players. Use the arrows to reorder.

Archived Players

▼

Players removed from the active roster who still appear in your game history. Restore them to make them selectable again in new games.

Base Games

▼

The DC Deck-Building Game sets you own. Each game session uses one base game. Check Rivals for sets that use the Rivals rules — Rivals sessions are limited to exactly 2 players.

Crossovers / Expansions

▼

Expansion packs that can be paired with a base game each session. Check Crisis for crossovers that use the cooperative Crisis rules — Crisis sessions replace individual scores with a shared team win/loss result.

Saved Additional Cards

▼

Promo cards and other extras that can be added to a game session via the Add Game screen. Name = the card's name (e.g. "Batman Ninja"). Type: Promo = official promotional hero cards; Other = any other extra card.

Saved Oversized Cards

▼

The large hero or villain cards each player selects at the start of a game. Each card must be assigned to the set it came from so it can be filtered in the Add Game screen.

Data

▼

All data is stored locally in your browser. Use Export to back up your data or transfer it to another device.
