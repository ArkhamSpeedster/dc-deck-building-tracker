DC Deck-Building Game Tracker

Track every game, every hero, every nemesis defeated.

What is this?

A personal tracker for the DC Deck-Building Game by Cryptozoic Entertainment. Log your sessions, track which heroes you play, how many nemesis cards you defeat, and see your performance over time — all saved locally in your browser with no server or account required.

Whether you're playing a base set, an expansion crossover, or Crisis mode, this tool keeps a detailed record so you can look back at your full play history and stats.

Compatibility note: Supports the standard competitive DC Deck-Building Game format. Does not yet support solo play or the Rebirth DC Deck-Building Game.

Getting started

  1. Go to ⚙️ Settings and add your player names (min 2, max 5).
  2. Add any base games, crossovers, and oversized cards you own. Batman and Superman are included by default.
  3. Head to 🎲 Log Game and record your first session — pick a base game, crossover, date, players, and oversized cards.
  4. Check 📊 Game Stats and 📜 Game History to track performance over time.

All data is stored in your browser's localStorage. Use Settings › Data › Export JSON to back up or transfer your history.

Features

  • Log Games — Base game, crossover, date, players, oversized cards, scores (VPs), and nemesis defeats per session
  • Additional Cards — Track promos and other extras per session from a saved library, filtered by type
  • Crisis Mode — Cooperative crossovers with team win/loss and a shared nemesis count
  • Rivals Mode — 2-player-only sets flagged in Settings; enforced on the Log Game screen
  • Game History — Filter by base game or crossover; sort by #, date, game, crossover, or type; edit or delete any entry
  • Player Stats — Win rates, average score, nemesis stats, most-used oversized cards, and Chart.js bar charts; compare two players side by side
  • Overall Stats — Aggregate game counts, most-played sets, top oversized cards, and crisis win rates
  • Archive & Restore — Archive players, games, crossovers, and cards without losing history; restore from a single combined Archived section
  • Ban & Unban — Mark oversized or additional cards as banned so they can't be picked in new games; labelled in history and stats
  • No build step — Plain HTML, CSS, and vanilla JS; open index.html directly in any modern browser

⚙️ Settings

Active Players

▼

Your player roster — these names appear in the Log Game form. Min 2, max 5 players. Use the arrows to reorder.

Base Games

▼

The DC Deck-Building Game sets you own. Each game session uses one base game. Check Rivals for sets that use the Rivals rules — Rivals sessions are limited to exactly 2 players.

Crossovers / Expansions

▼

Expansion packs that can be paired with a base game each session. Check Crisis for crossovers that use the cooperative Crisis rules — Crisis sessions replace individual scores with a shared team win/loss result.

Saved Additional Cards

▼

Promo cards and other extras that can be added to a game session via the Log Game screen. Name = the card's name (e.g. "Batman Ninja"). Type: Promo = official promotional hero cards; Other = any other extra card.

Saved Oversized Cards

▼

The large hero or villain cards each player selects at the start of a game. Each card must be assigned to the set it came from so it can be filtered in the Log Game screen.

Archived

▶

Items removed from the active lists. Restoring an item adds it back without affecting game history.

Banned Cards

▶

Cards banned from play because they may be game-breaking. Banned cards cannot be selected in new games and are labelled in history and stats. Unban to restore them to the active list.

Data

▼

All data is stored locally in your browser. Use Export to back up your data or transfer it to another device.
