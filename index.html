DeckLedger

Beta · Version 0.9.1-beta

An unofficial companion and statistics tracker for the DC Deck-Building Game created by Cryptozoic Entertainment.

Track the table

Log sessions, scores, oversized cards, nemesis defeats, Crisis outcomes, Rivals games, Multiverse variants, and extra cards in a clean responsive tracker. Base-game variant rules are not specifically modeled yet.

Keep backups

DeckLedger uses browser localStorage only. Browser cleanup, private mode, different devices, or clearing site data can erase saves, so export JSON backups regularly.

Fan-made

This is an unofficial, free companion tool. It may contain bugs, and exported/imported JSON compatibility is not guaranteed across all future versions.

Getting started

  1. Go to Settings, open the collapsed sections you need, and add your player names.
  2. Add your players and any oversized or additional cards you want to track. Base games and expansions are maintained by DeckLedger updates.
  3. Head to Log Game and record your first session — pick a base game, crossover, date, players, and oversized cards.
  4. Check Game Stats and Game History to track performance over time.

All data is stored in your browser's localStorage. Use Settings › Data Management to export backups, import JSON, or load sample data. Only import JSON files you trust.
