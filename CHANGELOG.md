# Changelog

## 1.0.0 - 2026-07-07

- Public release polish for DeckLedger.
- Added local Chart.js bundling so charts do not load from a CDN.
- Added Content Security Policy meta tag for static hosting.
- Added import preview, stricter JSON import validation, and clearer import warnings.
- Added visible app/export format version details in Data Management.
- Added backup reminders after every 5 logged games.
- Improved Settings organization, mobile readability, and collapsed sections.
- Added labels for archived, banned, and removed content in history/stats.
- Added sample data loader and demo JSON.
